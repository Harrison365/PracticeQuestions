# Greet Friends!

The `greetFriend` function will take a string of a name, and should return a string greeting them as in the examples below.

## Examples:

```javascript
greetFriend("Douglas");
// returns 'Hello Douglas!'

greetFriend("Maddie");
// returns 'Hello Maddie!'

greetFriend("Poonam");
// returns 'Hello Poonam!'
```
